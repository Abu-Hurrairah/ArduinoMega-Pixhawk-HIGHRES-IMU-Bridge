# Mission Planner Run Notes

The recorded lab session was converted into lightweight GitHub preview assets instead of uploading the full video file.

The visuals show:

1. Arduino Mega / sensor hardware connected.
2. Mission Planner running with the Pixhawk connection active.
3. Mavlink Inspector opened.
4. `HIGHRES_IMU (#105)` visible in the message list.
5. IMU fields such as acceleration, gyroscope, magnetometer-related fields, temperature, and timestamp visible in the inspector.
